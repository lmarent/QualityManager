Detailed Description

schedule timed events and execute the corresponding function at the correct time

The EventScheduler's task is to schedule and execute timed events in the meter system such as timed addition/removal of tasks (rules). A number of different events that are derived from the basic event class can be put into the EventSchedulers event queue.
