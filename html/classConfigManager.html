check if item is configured to be false

this will lookup the given item in the list of configured items and return true if the item is either set to be '0', 'false' or 'no'. Items which are configured but do not have a value are considered false.

  • name - name of queried item
    Returns
    a value !=0 if item is configured to one of '0','false','no'
