Detailed Description

provides logging of text messages to logging channels (files)

the Logger class is used to hold a number of FILE* for files in which logging of messages is to be done
so you only need to have one global Logger object lying around

usage:
create a Logger object
assign a logging channel (number) to a file (filename)
log messages to that channel (fprintf like syntax)

assign a channel number to a file

the file will be opened for appending text and each message logged to that channel will be written to that file
the file will be opened for appending in line buffered mode

  • prefix - text to prepend to each message logged via this channel
  • name - name of file to appends messages to (optional), if unspecified default log file is used
  • newlog - remove log file first if =1, else append to file
